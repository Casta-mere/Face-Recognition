# Face-Recognition
  
实验室人脸识别考勤系统
  
# Functions Description

- 录入

    通过某种前端界面录入人脸

- 识别
    
    摄像头拍摄视频，服务器端通过网络接口读取内容，进行人脸识别，识别出后发送数据至数据库。

- 导出

    通过某种前端界面导出考勤数据
